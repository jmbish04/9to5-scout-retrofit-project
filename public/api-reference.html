    üîê Authentication Required

    All API endpoints (except /api/health) require Bearer token authentication:

    Authorization: Bearer <token>

    üéØ Core Job Discovery & Management

    GET /api/health

    Health check endpoint (no authentication required)

    GET /api/jobs

    List jobs with filtering (status, source, limit)

    Query Parameters: status, source, limit, offset
    GET /api/jobs/{id}

    Get single job with complete details

    POST /api/runs/discovery

    Start discovery run

    curl -H "Authorization: Bearer <token>" -X POST /api/runs/discovery
    GET /api/agent/query

    Semantic job search using AI

    Query Parameter: q (search query)
    curl -H "Authorization: Bearer <token>" "/api/agent/query?q=senior%20python%20developer"

    üìà Job Monitoring & Analytics

    GET /api/jobs/{id}/tracking

    Get complete job tracking timeline with snapshots and changes

    GET /api/jobs/{id}/snapshots/{snapshotId}/content

    Download job snapshots (HTML, PDF, Markdown, Screenshots)

    Query Parameter: type (html, pdf, markdown, json, screenshot)
    POST /api/monitoring/daily-run

    Manually trigger daily monitoring workflow

    GET /api/monitoring/status

    Get monitoring status and market analytics

    üìö Job History & Applicant Management

    POST /api/applicant/history

    Submit job history for AI processing

    {
  "user_id": "user-123",
  "raw_content": "Senior Software Engineer at TechCorp (2020-2023)...",
  "content_type": "text/plain"
}
    GET /api/applicant/{user_id}/history

    Get applicant profile and structured job history

    POST /api/applicant/job-rating

    Generate AI-powered job fit rating (1-100 scale)

    {"user_id": "user-123", "job_id": "job-456"}
    GET /api/applicant/{user_id}/job-ratings

    View all job ratings for an applicant

    üìß Email Integration & Insights
