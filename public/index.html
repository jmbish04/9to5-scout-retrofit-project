Quick Start

Get up and running with 9to5-Scout in minutes.

Start Job Discovery 

# Start comprehensive job discovery
curl -H "Authorization: Bearer <token>" \
     -X POST https://your-worker.workers.dev/api/runs/discovery

# Semantic job search with AI
curl -H "Authorization: Bearer <token>" \
     "https://your-worker.workers.dev/api/agent/query?q=senior%20python%20developer%20remote"

# Generate AI-powered cover letter
curl -H "Authorization: Bearer <token>" \
     -H "Content-Type: application/json" \
     -X POST https://your-worker.workers.dev/api/cover-letter \
     -d '{"job_title": "Senior Software Engineer", "company_name": "TechCorp"}'
